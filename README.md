# Impuesto al Día - Landing

Landing page para el proyecto Impuesto al Día.

## Estructura
- `assets/`: Imágenes, logos, y otros recursos estáticos.
- `pages/`: Diferentes páginas de la aplicación.
  - `home/`: Archivos de la página principal (`index.html`, `style.css`, `script.js`).
- `index.html`: Redirección/Entrada al home.
